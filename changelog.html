You can access CustomTube's settings by clicking on the icon in the browser toolbar. Just look for this icon!

Changelog:

Version 1.0.3 - May 24, 2023

- Improvement: Like and dislike buttons are now green and black respectively [Affects Early 2013 and 2013 layouts]

- Improvement: The watch page is now even more accurate to the original layout [Affects all layouts from 2012 to 2014]

- Improvement: Video thumbnails that appear once the video ends are now unrounded [Affects all layouts, excluding the unmodified current layout]

- Improvement: Dropdown menus are now unrounded [Affects all layouts, excluding the unmodified current layout]

- Change: You can now see CustomTube's version number on the Configuration Window's topbar

- Fix: Fixed a bug where the topbar stayed white while in theater mode, when it was supposed to become dark [This bug affected the 2019 layout]

- Fix: Fixed a bug where the topbar icons turned white while in theater mode, when they were supposed to remain dark [This bug affected the 2014-2017 layouts]

- Other: In response to a new change YouTube made, playlist thumbnail styling is temporarily disabled (Mix playlist thumbnails are unaffected) [This affects all layouts from 2012-2017]

Version 1.0.2 - May 21, 2023

- New: Added "Smaller" option to "Related Videos Size". [Compatible with all layouts]

- Improvement: A couple of minor layout and CSS improvements [Affects all layouts to varying degrees, most notably the Early 2013-2014 layouts]

- Improvement: Using the site should feel a bit faster and smoother overall

- Improvement: If the comments haven't loaded after 5 seconds, the extension now forces them to

- Fix: Fixed a major bug where the performance would gradually degrade as you used the site [This bug affected all layouts]

- Fix: Fixed a bug where the Home Button text would not change according to the user's settings in certain situations (always happened when signed out, but could still happen when signed in) [This bug affected all layouts]

- Fix: Fixed a bug where the video wouldn't resize to fit inside the miniplayer [This bug affected the 2012 layout]

Version 1.0.1 - May 16, 2023

- New: Added option to make the "C" in "My Channel" lowercase on the My Channel button [Compatible with all layouts]

- Improvement: A couple of very minor CSS improvements [Affects 2013 and 2014 layouts]

- Fix: Fixed a major bug where the watch page wouldn't load correctly if YouTube's langauge was set to something other than English [This bug affected all layouts from Early 2013 to 2017]

- Fix: Fixed a minor bug where the Share button (the one located in the video player itself) was invisible [This bug affected all layouts from Early 2013 to 2017]

- Fix: YouTube's experimental new icon system was causing fairly serious issues, now they are fixed [This bug affected all layouts, excluding the unmodified current layout]

Version 1.0.0 - May 12, 2023

- First version of CustomTube! Notable features include choosing between the 2012-2017 and 2019 layouts, and being able to loop videos by default.
