Version 1.2.0 - October 4, 2023

- The "tabs" permission has been removed in the Chrome version.

- New: Added 2011 Layout option!

- New: Added Late 2009 Layout option!

- New: Added 2008 Layout option!

- New: Added option to choose the video player style. The style is chosen by your selected layout by default (All layouts from 2012 and eariler use the 2012 player as of now). There is also an option to use the current video player should you have issues with the player styles.

- Improvement: Player is now 640x360 instead of 586x330 [This affects the 2012 layout]

- Improvement: Slight improvement to the Search Page [This affects the 2012 layout]

- Improvement: The channel description now uses smaller text [This affects all layouts from 2017 and eariler]

- Improvement: More of the old watch page icons have been restored [This affects the 2019 and 2020 layouts]
